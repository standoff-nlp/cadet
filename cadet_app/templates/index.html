Welcome!

Cadet is an annotation application that lets you add complex labels with values. If you're working in TEI, for example, you can add a person tag as well as attributes for age, gender and so on. Cadet can also help you create a Univeral Dependencies treebank with output in CoNLL-U. If you've ever wanted to get your friends and students to spend hours annotating the texts you love - this is you. We also value everybody's time, so we've added machine suggestions based on existing annotations. Also if you've labeled it once, wouldn't it be nice to update the same thing wherever it appears in the text? Cadet does that because there are other things to do with text than annotation. We hope that Cadet works for you. Chao!
