Welcome!

Cadet is an annotation application that lets you add complex labels with attributes. If you're working with TEI, for example, you can add person tags as well as attributes for age, gender and birthplace. Cadet can also help you create Univeral Dependencies treebanks with output in CoNLL-U. If you've ever wanted to get your friends to spend hours annotating the texts you love - this is you. We value everybody's time, so we've added machine-generated suggestions based on existing annotations. Also, wouldn't it be nice to update the same thing wherever it appears in the text? Cadet does that because there are other things to do with text than annotation. We hope that Cadet works for you. Chao!
