A label set is an ordered set of label groups. This setting determines what label groups are visible in the annotator view and their order in the table.

Label Group

A label group is a cluster of associated labels. For example, part of speech or lexical features

Label

A label has a name and can be associated with multiple attributes. Each attribute has a key and value. We could have a PERSON label, for example, with attributes 'sex='intersex', 'role'='god' and 'age'='imortal'
